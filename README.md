# Ecommerce-data-analytics-and-visualization
Dashboards built using Tableau for Amazon, Olist, and Walmart dataset analysis

# Overview
As part of the internship, a mini project was undertaken to analyze three e-commerce datasets of a common theme and perform data analysis, visualization, and insight generation. The selected theme was e-commerce sales, using Amazon Sales and Reviews, Olist Brazilian E-commerce, and Walmart Supermarket Sales datasets.

The objective was to analyze customer behavior, sales trends, operational efficiency, and delivery performance, and present meaningful insights through interactive dashboards.


# Objective
To analyze different aspects of the e-commerce business and generate data-driven insights to improve decision-making, customer satisfaction, and operational efficiency.

---

# Datasets Used
- Amazon Sales and Reviews  
- Olist Brazilian E-commerce Dataset  
- Walmart Supermarket Sales Dataset  

# Analysis Performed

# Amazon Dataset
- Customer behavior analysis  
- Product ratings and reviews study  
- Pricing and sentiment analysis  

#  Olist Dataset
- Delivery performance analysis  
- Order delays and seller efficiency  
- Impact of delivery on customer satisfaction  

# Walmart Dataset
- Sales trend analysis  
- Seasonal pattern identification  
- Store performance and forecasting  

# Key Insights
- Pricing and product quality strongly influence customer satisfaction  
- Delivery delays directly impact customer ratings  
- Seasonal demand affects sales patterns significantly  
- Improving logistics and forecasting improves overall efficiency  

# Tools Used
- Excel  
- Tableau  

# Outcome
This project improved understanding of real-world e-commerce systems and strengthened skills in data analysis, visualization, and insight generation using industry datasets.
